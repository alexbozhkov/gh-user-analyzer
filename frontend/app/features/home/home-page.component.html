REST Path

Backed by your FastAPI REST endpoint and GitHub REST upstream calls.

  • Token is optional
  • Great for assignment-style REST output
  • More upstream requests when repository language details are expanded
 GraphQL Path

Backed by your Strawberry API and GitHub GraphQL pagination.

  • GitHub token is required
  • Single shaped query with pagination support
  • Ideal for comparing rate-limit efficiency against REST
